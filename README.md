To run the backend server:

   ```bash
   cd part3/phonebook-backend
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the server:

   ```bash
   npm run dev
   ```

Make sure you have a `.env` file with the correct MongoDB URI and port configuration.
